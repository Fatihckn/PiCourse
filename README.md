# PiCourse

Bu proje, öğrenciler ile eğitmenleri buluşturan basit bir MVP (Minimum Viable Product) özel ders platformudur.

## Kullanılan Teknolojiler

**Backend:** Django + Django REST Framework  
**Mobile:** Flutter  
**Database:** SQLite  
**Authentication:** JWT
